How to Play

Game Rules

1. Zwin WinGo is a 30-second lottery game. A new round starts every 30 seconds

2. You can bet on Color (Green/Violet/Red), Number (0-9), or Big/Small.

Color Rules

Green: Win if result is 1,3,7,9. (Pays 2x)

Violet: Win if result is 0,5. (Pays 2x)

Red: Win if result is 2,4,6,8. (Pays 2x)

Number Rules

Bet on any number 0-9. If your number matches, you win 9x your bet!

Big/Small Rules

Big (5-9): Win if result is 5,6,7,8,9. Pays 2x.

Small (0-4): Win if result is 0,1,2,3,4. Pays 2x.

Min/Max Bet

Minimum bet: Rs 1
